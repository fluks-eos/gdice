Description
===========

A dice program with various different dices for Elementary OS.

Dependencies
============

Building the program requires GTK+3, flex and bison. gstreamer-1.0 is optional.

Install
=======

```
./autogen.sh
./configure
make
make install
```

Uninstall
=========

```
make uninstall
```
